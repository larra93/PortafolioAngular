Otecscom

Proyecto realizado a la empresa Otecscom el cual consistia en desarrollar un nuevo sitio web corporativo, con la posibilidad de vender sus cursos en línea. Para ello se hizo uso de la plataforma flow.cl para el proceso de compra. Mientras que para el sitio se utilizo como base el framework Laravel. El sitio es totalmente administrable por lo que los administradores de otecscom pueden registrar, editar y modificar el contenido del sitio.

Santa María de Belén

Sitio web corporativo para el colegio Santa Maria de Belén de Coquimbo. Desarrollado en Wordpress, en el cual se suben las noticias correspondientes del colegio.

Detalles de amor

Sitio Web corporativo para pyme Detalles de amor, dedicada al arreglos de globos, desayunos y arreglos. Desarrollada con PHP nativo , javascript, css, Html5, usando como base de datos, Mysql.
